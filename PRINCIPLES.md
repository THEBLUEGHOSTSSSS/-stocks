# 美股量化半自动助手原理说明

## 1. 定位

这套系统是一个本地运行的半自动研究与执行辅助器，不是自动交易系统。

它的目标是把公开市场数据压缩成五类结果：

1. 当前市场环境是否允许增加风险暴露。
2. 你的已有多头和空头持仓应更偏加仓、持有、收缩还是回补。
3. 预设股票池里是否出现值得跟踪的新 Alpha 目标。
4. 在账户预算、借券条件和保证金约束下，哪些建议具备执行条件。
5. 在当前波动环境下，期权结构该偏买波动还是卖波动。

系统只输出分析、门槛和执行参考，实际下单始终由使用者自己完成。

## 2. 设计原则

当前版本遵循四个基本原则：

1. 先门控，后方向：先问“能不能做”，再问“往哪边做”。
2. 先数据可信，后模型可信：坏数据优先熔断，不让脏点进入下游。
3. 先账户约束，后理想收益：预算、保证金和借券条件优先级高于模型分数。
4. 先波动环境，后期权结构：期权模块先判 IV 与 IVR，再决定结构。

## 3. 系统分层

当前实现可以理解为六层：

1. 数据层：抓取行情、新闻、宏观、期权和借券数据。
2. 校验层：对历史行情、实时报价、成本价做 sanity gate。
3. 特征层：计算 RSI、ATR、历史波动率、成交量比率、相对强弱等指标。
4. 状态层：判断市场处于单边上涨、宽幅震荡还是系统性风险。
5. 交易层：计算预期收益、信号模式、EV、胜率和门控约束。
6. 输出层：在页面上展示指数信号、持仓执行单、期权预言机、异动雷达，以及 Markdown / JSON 报告。

## 4. 数据层原理

### 4.1 行情与回退逻辑

行情优先来自 yfinance；若海外源不可达、TLS 握手失败或返回空结果，会自动回退到东方财富。

分析入口会统一抓取：

1. 核心指数信号标的：SPY、QQQ、DIA、IWM。
2. 市场状态辅助标的：QQQ、NVDA、AAOI。
3. 用户实际持仓代码。
4. 异动雷达与图谱套利所需的扩展股票池。

日线主要用于指标、状态与回测；扩展时段价格单独用于盘前、盘后和夜盘展示。

### 4.2 数据熔断网关

当前版本不再默认相信任何上游价格输入。数据进入因子层前会经过统一校验：

1. 若单日绝对涨跌幅超过阈值，直接判定为异常历史数据。
2. 若日内高低振幅异常放大，阻断该段历史进入指标计算。
3. 若实时报价相对参考价偏移过大，则降级为空报价并输出告警。
4. 若成本价与当前市价偏离过大，则按现价熔断重置，避免持仓盈亏失真。

这一层的目标不是“修正数据”，而是“阻止坏数据污染结论”。

### 4.3 宏观与新闻

宏观层当前使用：

1. VIX。
2. 10 年期美债收益率。
3. ETF 活跃度代理。

新闻层当前主要解析标题，不做全文语义理解，重点提取：

1. 情绪方向。
2. 财报、指引、监管、收购等事件权重。
3. 标题密度与时间衰减。

### 4.4 期权数据层

期权数据层现在已经不是“成交量摘要”而是“两条路径并行”：

1. 期权流摘要路径：汇总近月 Call / Put 成交量、量比和 ATM 链概况，用于页面展示和方向辅助。
2. 期权定价路径：抓取近月 ATM 窗口的完整 Call / Put 链，用市场价格反解 IV，并写入本地 52 周 ATM IV 缓存，用于 IV Rank 与期权预言机。

当前近月定义为首个满足 7 到 45 DTE 的到期日。这样做是为了跳过 0DTE 和超短到期链上的极端噪声。

## 5. 特征与现货信号原理

对每个被分析标的，系统会构建一组标准化技术快照，核心包括：

1. 日涨跌幅。
2. 10 日或更长周期的趋势特征。
3. 20 日历史波动率。
4. RSI14。
5. ATR14。
6. 20 日成交量比率。
7. 均线与相对强弱。

现货主逻辑当前强调“趋势过滤均值回归”，而不是纯超卖抄底：

1. RSI 偏弱只提供观察信号。
2. 长期趋势未破坏时才允许低吸。
3. 跌破关键趋势线或触发接飞刀风险时，直接强制观望。

## 6. 市场状态机原理

市场状态分数初始值为中性，然后根据一组加减分规则调整：

1. VIX 低时加分，高时减分。
2. 利率冲高压制风险资产，利率回落释放风险偏好。
3. ETF 活跃度上行时提升风险承受度。
4. QQQ、NVDA、AAOI 的结构状态会影响成长主线和风险代理判断。

最终把分数裁剪到 0 到 1，并映射成三类环境：

1. 高风险偏好。
2. 中性震荡。
3. 系统性风险。

页面里的市场状态、总建议风险敞口和很多门控阈值都来自这里。

## 7. 数学增强原理

### 7.1 SVD 正交化

横截面因子先做标准化，再用 SVD 提取正交主轴，目的是减少强相关特征在同一时点重复计分的问题。

输出包括：

1. orthogonal alpha score。
2. 原始特征权重。
3. 奇异值和解释度。

### 7.2 PCA 残差均值回归

收益矩阵做 PCA 分解，把系统性部分和特质部分拆开：

1. 共性收益用于观察市场主成分。
2. 残差收益用于寻找特质偏离。
3. 对残差做滚动 Z 分数，识别适合做回归的标的。

### 7.3 图谱套利

图谱套利不再只看单票突破，而是把：

1. 杠杆 ETF。
2. 主题 ETF。
3. 龙头与跟随标的。
4. 跨市场映射关系。

放进同一个关系图里，结合先行滞后相关、协整和冲击扩散识别候选。

### 7.4 学术 Alpha 摘要

系统会抓取近期 arXiv q-fin 和 NBER 工作论文主题，把研究方向压缩成：

1. signal family。
2. factor formula 草图。
3. entry / exit rule。
4. risk filters。

它不是直接交易模型，而是研究导航层。

## 8. 期权引擎原理

### 8.1 近月 ATM 链构建

对每个标的，期权数据层会：

1. 选择首个 7 到 45 DTE 的近月到期日。
2. 围绕现价选取 ATM 窗口内的多档执行价。
3. 同时抓取 Call 与 Put。
4. 对每条腿优先使用中间价，若中间价不可用则退回 last / bid / ask。

### 8.2 定价与隐波反解

当前期权定价和隐波反解基于 Black-Scholes-Merton：

$$
C = S e^{-qT} N(d_1) - K e^{-rT} N(d_2)
$$

$$
P = K e^{-rT} N(-d_2) - S e^{-qT} N(-d_1)
$$

并使用 Vega 作为导数项，通过 Newton 迭代优先反算 IV；若 Newton 不收敛，则使用 bracket fallback。

这样做的原因是：Yahoo 链上自带的 impliedVolatility 字段在近月、短到期和低流动性执行价上经常失真，不能直接作为 IVR 的历史基础。

### 8.3 ATM IV 与 52 周缓存

近月 ATM 窗口里的多条腿会被按“离现价距离 + 成交活跃度 + 价格质量”加权，得到当前 ATM IV。

然后系统会把这一天的 ATM IV 写入本地缓存，只保留最近 52 周窗口。

因此，IVR 的长期目标是直接基于真实缓存历史，而不是代理带宽。

### 8.4 IVR 计算

IV Rank 定义为：

$$
IVR = 100 \times \frac{IV_{current} - IV_{low}}{IV_{high} - IV_{low}}
$$

优先级如下：

1. 若本地 52 周 IV 历史样本足够，直接使用真实历史高低点。
2. 若历史不足，则暂时退回基于实现波动率的代理带宽。
3. 若连实现波动率都不足，则再退回静态带宽。

### 8.5 策略映射规则

当前策略映射是硬约束，不是软提示：

1. IVR > 80：禁止净买入期权，只建议卖波动结构，如 Short Strangle、Iron Condor。
2. IVR < 20：优先建议 Long Straddle。
3. 中间区间：按持仓、趋势破坏、财报事件和风险情境在 Covered Call、Bear Put Spread、Iron Condor 等结构间切换。

### 8.6 动态盈亏图

策略盈亏图横轴不再使用固定价格范围，而是按：

$$
[S_0 \cdot e^{-IV\sqrt{T}},\; S_0 \cdot e^{IV\sqrt{T}}]
$$

自动缩放，并把执行价、盈亏平衡点和预期波动区间一起标在图上。

## 9. 资金与风控层

系统不会因为“方向看起来对”就默认允许开仓。资金与风控层会统一检查：

1. 账户总权益。
2. 单笔多头 / 空头仓位上限。
3. 凯利折扣。
4. 借券费率与可借股数。
5. 空头动态保证金。
6. 全局观望强约束。

若市场太弱、样本不足、EV 太差、借券不可用或预算不够，建议会被直接收缩为 HOLD / OBSERVE / IGNORE。

## 10. 输出层

当前页面与报告会输出：

1. 核心指数信号。
2. 持仓执行参考。
3. 期权流摘要与期权预言机。
4. SVD / PCA 数学结果。
5. 图谱套利候选。
6. 学术 Alpha 摘要。
7. Markdown 与 JSON 报告。

输出层的目标是“给人看懂并可复核”，不是“给机器自动成交”。

## 11. 当前限制

1. 公开数据源存在延迟、缺失和阶段性质量问题。
2. 52 周 IV 历史缓存依赖本地持续运行积累，不等于完整外部历史数据库。
3. 学术 Alpha 摘要是研究导航，不是直接信号源。
4. 图谱套利和残差均值回归属于辅助模式，仍需人工判断流动性、消息面和成交条件。
